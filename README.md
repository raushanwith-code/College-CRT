# 🎯 COLLEGE-CRT (Java Practice)

This repository contains my **Campus Recruitment Training (CRT)** preparation using **Java**.
It focuses on building strong **logic, problem-solving skills, and coding fundamentals** required for placements.

* 🔤 Basic Java Programs

* 🔁 Loops (for, while)
* 🔢 Arrays (Basic to Intermediate)
* 🔣 Operators & Expressions
* ⭐ Pattern Printing (Basic to Advanced)
* 🧠 Logical Problem Solving
* 🔁 Number & Character Patterns
